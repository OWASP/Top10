# OWASP Top 10:2025

OWASP Top 10:2025 リリースへようこそ。

OWASP Top 10 は、開発者やウェブアプリケーションセキュリティに関わる人々のための、標準的な意識啓発文書 (Awareness document) です。これは、ウェブアプリケーションに対する最も深刻なセキュリティリスクについての、広範な合意事項を表しています。

## 本リリースについて (About This Release)

これは OWASP Top 10 の **2025年** 版です。このバージョンには、最新のデータとセキュリティ動向に基づいた更新が含まれています。
