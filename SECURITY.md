# Security Policy

## Reporting a Vulnerability

The OWASP Top 10 project takes security issues seriously.

If you discover a security vulnerability related to this repository or its content, **please do NOT open a public GitHub issue**.

Instead, report the issue responsibly using one of the following methods:

- Email: security@owasp.org
- OWASP Security Disclosure: https://owasp.org/www-policy/operational/security-disclosure/

Please include as much information as possible to help us understand and resolve the issue quickly.

## Supported Versions

This repository contains documentation and educational material.
Security issues related to OWASP projects should follow the OWASP responsible disclosure process.

## Acknowledgements

We appreciate responsible disclosure and the efforts of the security community to improve OWASP projects.
